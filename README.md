# **Board Search**
  
검색 API를 연동한 게시판 프로젝트입니다.<br>
정렬, 더보기, 다크모드 등 사용자 인터랙션 중심으로 구현했습니다.
<br>
<br>

## 주요 기능
- 검색 API 연동
- 정확도순 / 최신순 정렬
- 더보기 기능
- 다크모드 지원
- 반응형 UI
<br>

## 기술 스택
- Vanilla JavaScript (ES6+)
- HTML / SCSS
- Gulp
<br>
